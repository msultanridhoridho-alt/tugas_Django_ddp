Home

Halo, nama saya Muhammad Sultan Ridho. Saya adalah mahasiswa Teknik Informatika yang tertarik pada pengembangan web dan teknologi.

Melalui website sederhana ini, saya ingin memperkenalkan diri sekaligus menampilkan hasil pembelajaran saya dalam membangun aplikasi web statis menggunakan Django Framework yang terintegrasi dengan Bootstrap untuk mendukung tampilan yang responsif (RWD).

Website ini terdiri dari beberapa halaman utama, yaitu:

  • Home — Berisi gambaran umum tentang profil dan minat saya di bidang teknologi.
  • About Me — Menampilkan riwayat pendidikan serta pengalaman organisasi yang pernah saya ikuti.
  • Gallery — Berisi dokumentasi kegiatan dan aktivitas yang pernah saya lakukan.

Saya berharap website ini dapat menjadi media untuk menampilkan identitas diri, pengalaman, serta kemampuan saya dalam pengembangan web, sekaligus sebagai pemenuhan tugas mata kuliah yang sedang saya tempuh.
